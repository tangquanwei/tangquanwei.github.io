LLM prefill 阶段通常 GEMM 很大，GPU 利用率较高。
LLM decode 阶段 batch 小、seq 每次只生成一个 token，很多计算更接近 GEMV 或小 GEMM，所以 GPU 利用率容易下降。这也是为什么 TensorRT-LLM、vLLM、continuous batching、paged attention 这些系统要特别优化 decode 阶段。
